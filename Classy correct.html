Classy correct

An Anki2 add-on

This add-on removes the explicit styling from the corrected answer and replaces it with the classes typedgood or typedbad. It is up to the user, but possible now, to set up the style for this in the card template.

This add-on relies on a change i suggested for Anki, but that has not been implemented. You have to use my ospalh-special branch for this add-on to do anything.

The important bits of the card template used were 

.typedbad {color: #dc322f;}
.typedgood {color: #859900;}

Other bits were 

.card{
  font-family: 'Demos LT';
  font-size: 24px;
  text-align: center;
  color: #657b83;
  background-color: #fdf6e3;
}
#corrected span:hover  { text-shadow: 1px 2px 5px #657b83; }

The basic color scheme in the example images is called Solarized.

The original style is to show the the text in black with red or green background, which may not look too good with color schemes that don’t use black text.
