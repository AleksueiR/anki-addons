Show colored stroke order diagrams

An Anki2 add-on

This is a work in progress that is intended to replace my Kanji stroke color add-on and possibly coexist with cayennes’ add-on, this one used on desktop clients, cayennes’ on AnkiWeb and mobile devices.

Goals

  • Use one xml file as the data base.
  • Extract svgs from that xml and colorize on the fly
  • Make it possible to use different colors at different times, by using an image as template.
  • Add other effects, such as shadows, or possibly animation, on the fly. Again, with different parameters at different times.
  • Use this as a normal template or to replace the svgs/images from cayennes’ add-on

Data set

This uses the KanjiVG kanji stroke order data set as the base.
